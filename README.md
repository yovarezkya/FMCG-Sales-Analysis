# FMCG-Sales-Analysis
Sales Analysis FMCG Dataset using Python,
Sumber Data : Kaggle

## Project Overview
Analisis data penjualan FMCG untuk mengidentifikasi:

- Top Performing Products
- Under Performing Products
- Sales Trend
- Seasonal Pattern
- Impact of Promotion

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statistics
- ExponentialSmoothing

### Top Product & Bottom Product
Product Yogurt menghasilkan revenue tertinggi.
Product Juice menghasilkan revenue tertinggi.

### Top Channel
E-commerce menghasilkan total penjualan yang lebih tinggi dibandingkan Retail.

### Promotion Impact
Terdapat indikasi bahwa strategi promosi tidak efektif dalam meningkatkan permintaan pelanggan.

### Seasonality
Peak Season penjualan terjadi pada bulan Juli .

## Business Recommendation
1. Optimalkan Persediaan Berdasarkan Pola Musiman
2. Perkuat Strategi Penjualan E-commerce
3. Tingkatkan Efektivitas Program Promosi
4. Tingkatkan Ketepatan Waktu Pengiriman
5. Maksimalkan Potensi Multipack
6. Fokus pada Produk dan Region Berkinerja Tinggi
